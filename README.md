一个简陋的函数代码备忘录，功能亟待完善中...
功能：
是一个代码备忘录，左侧边栏可以分类显示一些常用函数的基本结构，选择函数可以在右侧边栏查看函数用法，点击函数并ctrl+C可以复制函数进入剪贴板，以直接粘贴到自己的程序中
程序可以通过点击悬浮窗打开或关闭，也可以用快捷键（暂定ctrl+D）呼出和最小化
可以以json文件形式分类存储一些常用或冷门易忘的函数等，并写上基本用法，遗忘时可以ctrl+D呼出查看用法、或复制函数直接粘贴进代码里，省去翻资料或上网搜索的功夫

基于python语言编写

注意：
函数库放在程序同一个文件夹下
目前已经写好stm32的常用hal库备忘录库

曾经尝试过参考windows剪贴板快捷键呼出+点击程序外的地方就能隐藏的切换模式，但是没有成功，故程序中的模式A暂时没有用
已知bug：有悬浮窗模式，每次最开始启动程序时程序是最小化状态，必须点击一次悬浮窗将程序切换到前台然后才可以用快捷键（ctrl+D）切换，否则程序会卡死
